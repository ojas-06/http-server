[HTTP](https://en.wikipedia.org/wiki/Hypertext_Transfer_Protocol) is the
protocol that powers the web. 

1. Ensure you have `cmake` installed locally
2. Run `./your_program.sh` to run the server, which is implemented in
   `src/main.cpp`.
3. Optionally, run the `makefile` which will create the `server` executable file directly.
4. Just execute `./server`.
